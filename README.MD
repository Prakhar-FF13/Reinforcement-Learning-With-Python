# Reinforcement Learning:

The repository includes RL and DRL NoteBooks. The algorithms implemented:

1. Random Search.
2. Hill Climbing.
3. MDP.
4. Monte Carlo.
5. Temporal Difference Methods.
6. Cross-Entropy.
7. Deep Q-Learning.
8. Actor-Critic; (Advantage and Asynchronous).
9. Policy Gradient Methods.
